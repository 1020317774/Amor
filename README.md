## 技术栈

- [x] Spring Boot
- [x] Spring Security
- [x] JWT
- [x] Mysql
- [x] Mybatis
- [x] MyBatis-Plus
- [x] Flyway
- [x] Elasticsearch
- [x] Redis
- [x] Swagger
- [x] Lombok
- [x] Hutool
- [x] FastJson
- [ ] JustAuth


## 安装指导

- 克隆当前项目

- 修改`application.yaml`切换开发环境

- 修改数据库及redis参数

- 启动`Application`

- 访问[`http://127.0.0.1:10000`](http://127.0.0.1:10000)
