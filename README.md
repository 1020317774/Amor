## 使用指导

- 克隆当前项目到本地

- 创建本地数据库：`aurora`

- 修改`application.yaml`
  - `db`：username & password
  - `redis`：host & password

- 启动`AuroraApplication`

- 访问[`http://127.0.0.1:8000`](http://127.0.0.1:10000)

- API文档[`http://127.0.0.1:8000/swagger-ui.html`](http://127.0.0.1:8000/swagger-ui.html)
